Waarom heb ik dit geleerd van mijn portfolio?

Als je kijkt naar de dingen die ik van mijn portfolio heb geleerd snap ik dat je verbaast bent. Maar er is een goede reden voor. Ik zal dus even 1 voor 1 langsgaan waarom ik heb geleerd wat ik heb geleerd.

  1. Maak eerst een prototype
    2. Wat er bij mij fout ging is dat ik zo'n zin had in coderen en 1 plan had en dat dus meteen wilde gaan maken. Dit was een slecht idee aangezien ik een site had verdeeld in blokken van 100vh. Hierdoor maakte ik het heel lastig voor mezelf om nog de content erin te doen. Waar ik dus achter kwam is dat het echt heeeeel handig is om eerst een goed prototype te hebben.
  2. HTML, CSS en JS
    3. Ik heb dus 2 versie's gemaakt van mijn portfolio en hier allemaal verschillende dingen gedaan met HTML, CSS en Javascript en hier heb ik ook heel veel van geleerd.
  3. Begin op small screen
    4. Ik was daarnaast ook nog is in mijn design begonnen met het maken van desktop. Hierdoor werdt het echt een rot werk om dat nog weer responsive te maken voor desktop en ik ben dus met dit portfolio eerst begonnen met telefoon.
